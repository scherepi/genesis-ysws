genesis

build an operating system, get compute to run it on.

◈

genesis is a YSWS (You Ship, We Ship) program by Hack Club where high school students (you!!) build their own operating systems from scratch and earn single board computers to run them on.

i know building an operating system seems really scary, but i promise it's one of the most rewarding and interesting things you can do in computer science. you'll learn low-level programming, operating-system fundamentals, and get a better grasp of how complicated these machines that run our lives really are! i believe in you, and i know you've got this!
